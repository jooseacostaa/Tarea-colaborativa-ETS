# Team Task Manager

Console-based Java application for managing tasks.

It allows you to:
- Add tasks
- List tasks

Project developed collaboratively using Git and GitHub,
working with feature branches and a develop branch.

---

## Getting Started

### Requirements

- Java JDK 8 or higher
- NetBeans (recommended) or any Java IDE
- Git (for version control)

### Installation

1. Clone the repository:
-
- git clone https://github.com/your-username/your-repository.git
- 
2. Open the project in NetBeans (or your preferred IDE).

3. Build the project.

### Running the Application

Run the `Main` class.

The console menu will appear with the available options:
- Add task
- List tasks
- Exit

---

## Project Structure

- `Tarea.java` → Represents a task (description and completion status)
- `GestorTareas.java` → Contains the task management logic
- `Main.java` → Interactive console menu
- `javadoc/` → Generated project documentation

---

## Documentation

Javadoc documentation is generated using the NetBeans "Generate Javadoc" tool and included in the `javadoc` folder.

---

## Version Control Workflow

- `main` → Stable and functional version  
- `develop` → Integration branch  
- `feature-*` → Feature development branches
